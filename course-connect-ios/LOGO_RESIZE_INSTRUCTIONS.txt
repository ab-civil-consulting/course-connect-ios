========================================
TOOLS FOR RESIZING
========================================

Option 1: Online Tools (Easiest)
- https://www.iloveimg.com/resize-image (PNG)
- https://ezgif.com/png-to-webp (PNG to WebP conversion)
- https://icon.kitchen/ (Generate all app icons at once)

Option 2: Photoshop/GIMP
- Open source logo
- Image → Image Size → Enter dimensions
- Export as PNG or WebP

Option 3: Command Line (Windows PowerShell/Linux)
- Install ImageMagick: winget install ImageMagick
- Resize command: magick input.png -resize 1024x1024 output.png
- Convert to WebP: magick input.png output.webp
